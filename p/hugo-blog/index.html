通过hugo + GitHub Pages搭建个人博客

Sep 09, 2020
阅读时长: 1 分钟

建站记录

安装hugo

官网hugo安装方法

开启GitHub Pages

前往GitHub，创建名为GitHub名.github.io的仓库

创建hugo blog

  1. 在你想要放置blog源码的位置打开终端
  2. 创建了一个名为my-blog的站点

    hugo new site my-blog

  3. 将你中意的hugo主题安装到themes文件夹下（我选择的是Stack主题）

    cd my-blog/themes
    Stack主题官方安装配置教程

git上传

  1. 在my-blog根目录下运行hugo命令

    hugo

  2. 移动至生成的public文件夹下：

    cd public/

  3. 使用git推送至GitHub仓库中

    git init
    git add .
    git commit -m “2022.02.10”
    git remote add origin https://github.com/syschars/syschars.github.io.git
    git push origin master

访问GitHub Pages站点

访问 https://syschars.github.io/
