Android--控件

Android控件相关知识

Feb 10, 2022
阅读时长: 2 分钟

控件所有的属性不仅可以在xml文件中进行配置，也可以在在Java代码中设置。
所有控件的属性有自己默认值。
xml文件相当于一份构造表（类似对象的构造器），在app启动时加载xml文件初始化控件的相关属性。
Java代码中进行的设置会在程序运行时对控件进行更改，所以Java的修改会覆盖掉xml中的启动设置。

TextView

控件大小

android:layout_width="200dp"    控件宽度
android:layout_height="200dp"   控件高度
android:gravity="center"        内容放置位置

内容文本

android:text="@string/text_2"       文本内容
android:textColor="@color/black"    文本颜色
android:textSize="50sp"             字体大小
android:textStyle="bold"            字体加粗、斜体

文本阴影效果

android:shadowColor="@color/red"    阴影颜色
android:shadowDx="10"               阴影横向偏移
android:shadowDy="10"               阴影纵向偏移
android:shadowRadius="10.0"         阴影透明度

走马灯效果

android:singleLine="true"                       文本限制为一行显示
android:ellipsize="marquee"                     在哪里省略文本
android:focusable="true"                        获取焦点
android:focusableInTouchMode="true"             在触摸时聚焦
android:marqueeRepeatLimit="marquee_forever"    效果的持续次数
<requestFocus />                                放于末尾，在程序启动时启动走马灯效果

Button

控件大小以及内容文本效果都与TextView相同

android:layout_width="200dp"
android:layout_height="100dp"
android:gravity="center"

按钮背景及颜色

android:backgroundTint="@color/btn_color_selector"
android:background="@drawable/btn_selector"

在res资源文件夹下创建一个color文件夹，创建Resource文件btn_color_selector.xml用于存储按钮的颜色变化

<item android:color="@color/red" android:state_pressed="true"/>
<item android:color="@color/black"/>

在res中的drawable中创建Resource文件btn_selector.xml文件存放按钮背景的变化

<item android:drawable="@drawable/ic_baseline_accessible_forward_24" android:state_pressed="true"/>
<item android:drawable="@drawable/ic_baseline_accessible_24"/>

按钮事件

方法一

为button创建一个id
android:id="@+id/btn_1"

在onCreate方法中

Button btn_1 = findViewById(R.id.btn_1);
btn_1.setOnClickListener(new View.OnClickListener() {...});              监听短按事件
btn_1.setOnLongClickListener(new View.OnLongClickListener() {...});      监听长按事件
btn_1.setOnTouchListener(new View.OnTouchListener() {...});              监听触摸事件

方法二

为button添加绑定函数：
android:onClick=“lalalaClick”

在onCreate方法自动创建了一个函数
public void lalalaClick(View view)
等价于方法一中的setOnClickListener函数

EditText

android:hint                输入提示文本
android:textColorHint       输入提示文本的颜色
android:inputType           设置输入文本的类型
android:drawableLeft        设置图片的插入位置（左）
android:drawablePadding     设置图片与输入框之间的距离
android:paddingLeft         设置内容与边框的距离（左）
android:background          背景色

ImageView

android:src                 设置图片来源
android:scaleType           图片缩放类型
android:maxHeigth           最大高度
android:maxWidth            最大宽度
android:adjustViewBounds    调整View的界限

ProgressBar

android:max                 进度条最大值
android:progress            已完成进度值
android:indeterminate       设置为true时，进度条不显示精度
style="?android:attr/progressBarStyleHorizontal"    水平进度条

Notification

notificationManager = (NotificationManager) getSystemService(NOTIFICATION_SERVICE);
    if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.O) {           //通知渠道从Android 8.0才引入，创建通知渠道需要判断运行的系统是否为8.0及之后的版本
        NotificationChannel notificationChannel = new NotificationChannel(
                "lalala", "测试通知", NotificationManager.IMPORTANCE_HIGH);
        notificationManager.createNotificationChannel(notificationChannel);
    }
    Intent intent = new Intent(this, MyNotification.class);
    notification = new NotificationCompat.Builder(this, "lalala")   //channelId要与NotificationChannel对象的id相同，用于为通知选择通知渠道
        .setContentTitle("通知")                                    //通知标题
        .setContentText("lalalalalala")                             //通知内容
        .setSmallIcon(R.drawable.ic_baseline_notifications_24)      //通知的小图标，图标只有alpha图层，不能是有颜色rbg图
        .setLargeIcon(                                              //通知的大图标，bitmap图，要将普通的rbg图像转换为bitmap图
            BitmapFactory.decodeResource( getResources(), R.drawable.avatar))
        .setColor(Color.parseColor("#ff0000"))                      //通知小图标的颜色
        .setContentIntent(PendingIntent                             //点击通知后可以跳转想要前往的activity，参数是PendingIntent类型，要事先进行转换
            .getActivity(this, 0, intent, 0))                       //requestCode与flags可以随便设置
        .setAutoCancel(true)                                        //自动清楚通知
        .build();

删除自带的toolBar （res/values/themes.tml）

<style name="Theme.MyApplication" parent="Theme.MaterialComponents.DayNight.NoActionBar">

相关属性

AlertDialog

View dialogView = getLayoutInflater().inflate(R.layout.dialog_view, null);  //加载一个布局
AlertDialog.Builder builder = new AlertDialog.Builder(this);                //创建一个builder，用来构造AlertDialog的属性
builder.setIcon(R.mipmap.ic_launcher)                                       //设置图标
        .setTitle("对话")                                                   //添加标题
        .setView(dialogView)                                                //使用布局
        .setMessage("今天天气不错")                                          //设置消息
        .setPositiveButton("确定", new DialogInterface.OnClickListener() {  //确定按钮
            @Override
            public void onClick(DialogInterface dialog, int which) {
                Log.e(TAG, "onClick: 确定");
            }
        })
        .setNegativeButton("取消", new DialogInterface.OnClickListener() {  //取消按钮
            @Override
            public void onClick(DialogInterface dialog, int which) {
                Log.e(TAG, "onClick: 取消");
            }
        })
        .setNeutralButton("中间", new DialogInterface.OnClickListener() {   //中间的按钮
            @Override
            public void onClick(DialogInterface dialog, int which) {
                Log.e(TAG, "onClick: 中间");
            }
        })
        .create()                                                           //创建AlertDialog
        .show();                                                            //将AlertDialog显示出来
//create()与show()固定在链式调用的最后，且顺序固定，其他的属性配置的函数位置与顺序随意

PopupWindow

View popupView = getLayoutInflater().inflate(R.layout.dialog_view, null);

//第一个表示采用的布局，第二、三个表示布局的宽度与高度，最后一个表示焦点不在PopupWindow后自动消失
PopupWindow popupWindow = new PopupWindow(
        popupView,
        ViewGroup.LayoutParams.WRAP_CONTENT,
        ViewGroup.LayoutParams.WRAP_CONTENT,
        true);
Button btn = popupView.findViewById(R.id.btn_3);                                    //获取PopupWindow中的控件
btn.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View v) {
        Log.e(TAG, "onClick: pop");
        popupWindow.dismiss();
    }
});
popupWindow.setBackgroundDrawable(getResources().getDrawable(R.drawable.avatar));   //从Drawable设置PopupWindow的背景图片

//设置显示时的偏移位置，第一个参数为参照物，之后两个参数为x,y方向上的偏移量（可以不设）
popupWindow.showAsDropDown(view, 100, 100);
